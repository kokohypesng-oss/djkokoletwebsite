A mobile-first music streaming theme for DJ Kokolet (Kokohypes).

== Description ==

Koko Mobile Theme is a modern, mobile-first WordPress theme designed specifically for music streaming and DJ services. It features dark/light theme switching, an integrated music player, booking system, and a sharp, modern digital aesthetic.
